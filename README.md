Sistema de Gestión de Inventario:


Descripción:
Travalo es una aplicación desarrollada en Java para la gestión de inventarios, diseñada para simular el control completo de productos dentro de un sistema de almacenamiento.
Cuenta con una interfaz interactiva basada en cuadros de diálogo (JOptionPane), permitiendo al usuario gestionar productos desde su registro hasta su venta, integrando persistencia de datos y estructuras de datos personalizadas para optimizar el manejo de la información.


Objetivo del Proyecto:
Desarrollar un sistema capaz de gestionar inventarios de manera eficiente, aplicando estructuras de datos, persistencia en archivos y principios de programación orientada a objetos.

Tecnologías Utilizadas
Java
Java Swing (JOptionPane)
Manejo de archivos .txt


Funcionalidades Principales
- Gestión de Productos
- Registro de nuevos productos
- Actualización de información
- Eliminación de productos
- Visualización completa del inventario
- Control de Inventario
- Organización del almacenamiento mediante matrices
- Clasificación de productos por categorías
- Representación estructurada del sistema de inventario
- Gestión de Ventas
- Registro de ventas
- Control de productos vendidos
- Actualización automática del inventario
- Reposición de Stock
- Procesamiento de reposición de productos
- Gestión eficiente de disponibilidad
- Persistencia de Datos
- Almacenamiento en archivos .txt:
- Inventario
- Ventas
- Productos vendidos


Aspectos Técnicos Destacados
Programación Orientada a Objetos (POO)
Uso de listas enlazadas y nodos para gestión dinámica
Separación de responsabilidades en clases:
Inventario
Producto
Estructuras auxiliares
Simulación de matrices para organización física del inventario
Interfaz interactiva mediante JOptionPane


Estructuras de Datos Implementadas
Listas enlazadas → Gestión dinámica de productos
Nodos → Representación interna de datos
Matrices → Organización del almacenamiento
Archivos → Persistencia de información
